# ClaimsIQ – Intelligent Insurance Claims Platform

## Overview

ClaimsIQ is an enterprise-scale AWS Data Engineering and Generative AI platform that simulates a production insurance claims processing system.

The platform demonstrates modern cloud architecture, data engineering best practices, infrastructure as code, governance, streaming analytics, and AI-powered decision making.
